## Installation
Run the following command in the terminal 
```
git clone https://github.com/33Arsenic75/COP290_Assignment_1.git
```
Open the directory and run the make file as follows
```
make symbol='SBIN' years=3
```
This will give u the data for SBIN symbol stock for the previous 3 years. One may similarly use it for other stocks.
